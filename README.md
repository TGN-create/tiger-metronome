# Tiger Metronome

Master your chops with this sleek, web-based metronome! Designed for precision and ease of use, the Tiger Metronome is built entirely with vanilla web technologies, meaning you can run it right in your browser without any complex setup.

## Features
- **Precise Timing:** Reliable audio scheduling using the Web Audio API.
- **Adjustable Tempo:** Easily dial in your exact BPM from 20 to 280.
- **Custom Meter:** Support for a variety of time signatures (beats per bar).
- **Timed Session:** Includes a 2-minute countdown timer that automatically stops the metronome when time is up—perfect for focused practice bursts.
- **Keyboard Shortcuts:** Control the metronome easily using the Spacebar (Play/Stop) and Arrow Keys (Adjust BPM).

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla, Web Audio API)

## How to Run
Since this is a front-end only application, there's no build step required.
Simply open the `index.html` file in any modern web browser to start practicing.
